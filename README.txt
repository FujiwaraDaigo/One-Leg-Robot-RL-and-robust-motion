別途開発環境の構築が必要でこのまま動かすことはできません
研究の目的は，下記１のプログラムでロボットの動作をシミュレータ上で生成できているにも関わらず，それがモデル化誤差の影響により現実のロボットではうまく動作しないという問題に対して，学習方法を工夫することで解決を図るというものです．

１stabilize_karakasa.cpp
物理エンジンODEを使い，強化学習アルゴリズムPI２によって一脚ロボットの移動戦略を獲得するプログラムです．

２stabilize_karakasa_additional_2.cpp
１で学習した動作サンプルから，より環境誤差に強いロバストな動作を選び，その動きに合わせて価値関数を変更する

３stabilize_karakasa_reflection.cpp
変更した価値関数をもとに再学習を行い，よりロバストな動作を得る．

４stabilize_karakasa_movie.cpp
デバッグ用のシミュレーション動画作成プログラム
